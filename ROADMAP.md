- [ ] **باقات منتجات + Upsell/Cross-sell** — بيع منتجات مع بعض بخصم / اقتراح منتج مكمّل
- [x] ~~كوبونات الخصم~~ — موجودة أصلاً (discount_code / discount_amount بجدول orders)
- [x] ~~حملات واتساب تسويقية~~ — موجودة أصلاً (تذكير سلات متروكة بالجملة) — **لازم نوسّعها لحملة عامة لكل قائمة العملاء (مو بس سلات متروكة)** — مهم ومكرر بطلب عثمان، أولوية عالية
- [x] ~~تقييمات ومراجعات المنتجات~~ — موجودة أصلاً (تحقق فعلي بالكود قبل ما نبني أي شي فوقها)
- [ ] ~~برنامج نقاط ولاء~~ — مش مهم هلأ (مؤجل)

## 🥈 تحسينات جودة المنصة

- [ ] **صلاحيات موظفين متعددة (Multi-user roles)** — حسابات فرعية بصلاحيات محدودة لكل متجر
- [ ] **تحليلات على مستوى المنتج (Product Analytics)** — تحويل، مشاهدات، أداء كل منتج لحاله
- [ ] **مدونة/صفحات SEO (Blog)** — أداة تسويق عضوي جوا لوحة التحكم
- [ ] **حقول مخصصة (Custom Fields)** — مرونة إضافة معلومات خاصة لكل قطاع

## 🚀 أفكار أكبر (كلها ممتازة — تتنفذ لاحقاً بالترتيب المناسب)

- [ ] ودجت "عدد المشاهدين الآن" / إشعارات شراء لحظية (social proof)
- [ ] برنامج إحالة للعملاء (referral program)
- [ ] قناة اكتشاف مشتركة بين متاجر Mawq3i (شبيه "مزيد")
- [ ] واتساب بيزنس API رسمي (بدل روابط wa.me)
- [ ] كاشير/نقاط بيع (POS) — مشروع كبير لحاله

## 📌 بنود جديدة أضافها عثمان (11 يوليو 2026)

- [ ] **نسبة التحويل (Conversion Rate)** بلوحة التحليلات
- [ ] **عدد الزيارات (Visits/Traffic)** بلوحة التحليلات
- [ ] **تتبع أهداف** (زي أهداف زد بالتحليلات)
- [ ] **مزامنة المخزون أونلاين وأوفلاين** — التاجر يعرف قديش باع أونلاين وقديش باع بالمحل الفعلي، من نفس لوحة التحكم. **مهم جداً — هاي اللي بتخلي المنصة تُستخدم يومياً حتى جوا محل التاجر، مش بس أداة إدارة متجر إلكتروني.**
- [ ] **أعلى المدن مبيعاً** — عشان التاجر يركّز إعلاناته عليها
- [ ] **إضافة كل حسابات التواصل وأرقام الهاتف بالمتجر** (انستقرام، فيسبوك، تيك توك، واتساب، إيميل...) — تظهر بواجهة المتجر
- [ ] **الأسئلة الشائعة (FAQ)** بكل متجر
- [ ] **سياسة التبديل والاسترجاع** بكل متجر
- [ ] **مصادر الزيارات (Traffic Sources)** — من وين الزوار جايين (انستقرام، جوجل، مباشر...)
- [ ] **أفضل العملاء (Best Customers)** — تحليل بأعلى العملاء شراءً
- [ ] **لوجستيات متل زد** — نظام شحن/توصيل أشمل (Togo موجود، بس نطوّره ليكون أشمل زي حل زد شب)
