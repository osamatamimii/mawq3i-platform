ANIMATION SYSTEM (VERY IMPORTANT)
========================
Use Framer Motion:

1. Page transitions:
- Fade + slight slide up
- Duration: 0.4s

2. Cards:
- On hover: translateY(-4px) + shadow

3. Buttons:
- Scale 1.03 on hover
- Smooth transition

4. Tables:
- Row hover highlight

5. Scroll:
- Sections fade-in on scroll

6. Avoid:
- Heavy animations
- Lag
- Over-effects
